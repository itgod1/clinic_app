================================================================================
                    口腔诊所SaaS系统 - API接口文档
                         Version 1.0.0
================================================================================

【文档说明】
本接口文档包含四个部分：
  1. 患者端接口（小程序）
  2. 医生端接口
  3. 管理端接口（Web后台）
  4. AI模块接口

【通用说明】
- 基础路径：/api
- 请求格式：application/json
- 认证方式：Bearer Token (JWT)
- 响应编码：UTF-8
- 时间格式：yyyy-MM-dd HH:mm:ss（日期时间）、yyyy-MM-dd（日期）、HH:mm（时间）

【统一响应格式】
{
  "code": 200,           // 状态码：200成功，400参数错误，401未授权，403禁止，404未找到，500服务器错误
  "message": "success",  // 消息
  "data": {},            // 数据对象
  "timestamp": 1704067200000  // 时间戳
}

1.1 登录接口
────────────────────────────────────────────────────────────────────────────────

【用户登录】
POST /api/auth/login

请求参数：

      "userId": 1,
      "username": "admin",

────────────────────────────────────────────────────────────────────────────────
1.2 文件上传接口
────────────────────────────────────────────────────────────────────────────────

【上传文件】
POST /api/upload/image
Content-Type: multipart/form-data

请求参数：
- file: 文件流（必填）
- bizType: 业务类型（选填，默认为common）
  - patient：患者头像
  - medical：病历附件
  - doctor：医生头像
  - clinic：诊所资质

响应示例：
{
  "code": 200,
  "message": "上传成功",
  "data": {
    "url": "https://cdn.xxx.com/uploads/2024/04/15/xxx.jpg",
    "fileName": "xxx.jpg",
    "fileSize": 102400
  }
}

【上传多个文件】
POST /api/upload/images

请求参数：
- files: 文件流数组
- bizType: 业务类型

响应示例：
{
  "code": 200,
  "message": "上传成功",
  "data": [
    {"url": "https://cdn.xxx.com/1.jpg", "fileName": "1.jpg"},
    {"url": "https://cdn.xxx.com/2.jpg", "fileName": "2.jpg"}
  ]
}
